volApp is a Spring Boot project for managing flight information. It utilizes the Spring Boot framework and Maven as a build tool. 
Project Structure:
Dependencies:

Spring Data JPA: Implements Java Persistence API for data access.
Spring Web: Provides the necessary dependencies to build a web application.
Spring DevTools: Enables fast application restarts during development.
MySQL Connector/J: MySQL JDBC driver for database connectivity.

How to Build:

To build the project, use the following Maven command:

bash
Copy code
mvn clean install
How to Run:

After building the project, you can run the application using the following command:

bash
Copy code
mvn spring-boot:run
